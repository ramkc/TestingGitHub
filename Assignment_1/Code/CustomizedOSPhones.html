Customized OS phones contain customizations of the operating system of the smartphone. The Samsung Galaxy S series, HTC One series, are examples of this category of smartphones. The pros and cons of purchasing such smartphones are listed below.


Pros

Better Functionality
 More Choices


Cons

Slower Updates
 Doubtful Upgrades


Better Functionality


Customizations from handset manufacturers add a number of functionalities which are considered useful by a good number of people. These usually improve the usability of the smartphones and provide a better user experience.

More Choices


There are a larger number of handset models which have customized OS on them. There is greater variety in the handset models and price ranges - one can find a phone that fits a wide range of budgets.

Slower Updates


When a newer version of the operating system is released, manufacturers typically take time to customize the OS with their skins. Releases of the customized OS phones lag the release of the OS releases by many quarters.

Doubtful Upgrades


The phone is guaranteed to be upgraded, even if the hardware of the handset is capable meeting the requirements of the newer versions of the OS. Manufacturers typically ship upgrades to higher end series while lower end phones generally receive a raw deal with regard to upgrades.
