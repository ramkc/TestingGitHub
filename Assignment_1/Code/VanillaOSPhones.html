Vanilla OS phones contain no customizations of the core operating system of the smartphone. The IPhone, Google Nexus, Blackberry are examples of this category of smartphones. The pros and cons of purchasing such smartphones are listed below.


Pros

Faster Updates
 Higher Stability
Lower TCO
 Guaranteed Upgrades


Cons

Lower Functionality
 Limited Choices


Faster Updates


When a newer version of the operating system is released, manufacturers typically take time to customize the OS with their skins. Using a vanilla OS handset would allow getting the newer versions of OS faster on the handset.

Higher Stability


In the world of software, less is better - more code means more liability. The stability of vanilla OS smartphones is likely to be higher, compared to the customized ones which add a lot of code in order to achieve the additional functionality.

Lower TCO


The phone stays current for longer number of years. The need to upgrade the hardware in terms of buying newer handsets is reduced.

Guaranteed Upgrades


As long as the handset can support the newer versions of the OS, the phone is guaranteed to be upgraded. The problem of capable handsets not receiving upgrades because of the manufacturers not releasing customized versions of the OS for older handset is avoided.

Lower Functionality


Customizations from handset manufacturers add a number of functionalities which are considered useful by a good number of people. These usually improve the usability of the smartphones and provide a better user experience.

Limited Choices


The number of handsets which are available with vanilla OS are limited. On the other hand, there are a larger number of handset models which have customized OS on them.
